## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/parisa-portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## 🚀 Deployment

This project is deployed on Vercel.

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```
